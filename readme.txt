goropriをダウンロードいただきまして、ありがとうございます。

このソフトはまだ試験段階です。
動作不良などについて、個別の対応はいたしかねますが、報告していただけると、バージョンアップの際に改善できるかもしれませんので、よろしくお願いします。
また、本ソフトは、Windows10での環境を想定しています。他の環境でのテストはしていないので、不具合があれば直接上記の連絡先よりお問い合わせください。

実行ファイルの場所は、dist\goropriにあります。各自ショートカットを作ってください。（自動で作る方法がわからない）

基本セットについて
　最初の語呂を登録するときに基本セットを登録するかを選択しますが、
Help
・入力時のバグ
　数字は全て半角で入力してください。それでも起こるバグについては、お問い合わせください。

・語呂の削除方法
　ver0.1.1では、語呂の削除は実装されていません。そのため、実行ファイルと同じ場所にある"goro.csv"から削除する必要があります。その方法を示します。<注意>この操作は取り消せません。心配であれば、csvファイルのバックアップを取っておくことをおすすめします。
　goro.csvを右クリック、"プログラムから開く">"メモ帳"を選択
　削除したい語呂のある行を、行末の改行を含めて削除する
　"goro.csv"を保存して閉じる

・アップデートの方法
　新しいバージョンを利用する場合のデータの更新の方法を、以下に示します。語呂のデータは全てcsvファイルの中にあります。
　新しいバージョンをダウンロードします。
　古いバージョンのdist\goropriから、"goro.csv"をコピーします。
　新しいバージョンのdist\goropriに貼り付けます。もしこの場所がない場合、新しいバージョンの"readme.txt"にその場所、もしくは他の方法が書いてあります。
　新しいバージョンで動作を確認してください。確認できたら、古いバージョンは削除してもかまいません。

注意事項
　このソフトの二次配布を禁じます。
　このソフトを利用している画像などを公開することを許可します。
　このソフトを利用して生成された素数のどんな利用も許可します（許可するまでもないですが）。ぜひかわいがってください。
　このソフトを通したデータ収集などは行っていません。
　このソフトの利用を知らせていただけると、作成者が喜びます。
